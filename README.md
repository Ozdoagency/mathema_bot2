# Mathema Bot

Telegram bot integrated with Mistral AI.

## Функциональность

- Обычный чат с AI
- Fill-in-the-middle (FIM) генерация через команду /fim
- Эмбеддинги текста

- /fim - Fill-in-the-middle генерация
- /fimhelp - Помощь по использованию FIM
